UI Test Automation –ILcarro Web Project

Tech Stack
Java
Selenium WebDriver
TestNG
Gradle
Project Description
This project contains automated UI tests for ILcarro web application.
It covers core user workflows such as contact management and authentication.

Test Coverage:

User registration and login
searching cars
add new cars
Validation scenarios (positive & negative cases)
Framework Features
Reusable page components
Clear test structure with TestNG
Easy test execution with Gradle
Separation of test logic and UI elements
How to Run Tests
Clone the repository
Open the project in your IDE
Run the following command:
gradle test
📊 Purpose

The purpose of this project is to demonstrate hands-on experience in UI test automation, including:

Building a scalable automation framework
Writing maintainable test cases
Applying best practices in test design
Working with modern automation tools
